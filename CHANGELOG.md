# Changelog

## 0.0.1

- Initial release of `native_in_app_purchase`
- Added native Flutter plugin architecture for Android and iOS
- Added `MethodChannel` API for initialize, product queries, purchase flow, restore, and purchase completion
- Added `EventChannel` purchase update stream
- Added Android implementation using Google Play Billing
- Added iOS implementation using StoreKit
- Added `NativeInAppProduct`, `ProductQueryResponse`, `Purchase`, `PurchaseParam`, and `PurchaseVerificationData`
- Added consumable support and pending purchase completion flow
- Added working example app
- Added tests and analyzer coverage
